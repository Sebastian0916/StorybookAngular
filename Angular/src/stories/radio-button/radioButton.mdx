# RADIO BUTTON

Los radio button (botones de opción) de Angular Material son un tipo de componente de selección que permite al usuario elegir una única opción de un conjunto de opciones disponibles. Estos botones son ideales para presentar opciones mutuamente excluyentes en una interfaz de usuario, donde el usuario solo puede seleccionar una opción. Los radio button de Angular Material ofrecen diferentes opciones de configuración, como el estilo, el tamaño y la posición en pantalla, lo que los hace muy flexibles y adaptables a diferentes diseños y necesidades. Además, los radio button pueden ser fácilmente integrados en una aplicación Angular utilizando la biblioteca de componentes Angular Material, lo que facilita su implementación y personalización.
