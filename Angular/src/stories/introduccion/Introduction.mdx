# Angular Material para SincoSoft

Angular Material es la librería de componentes de interfaz de usuario de Angular que usamos para implementar diseños en Sincosoft.
Originalmene esta librería la implementa Material Design de Google.

<button color="primary">
  [Documentación Angular Material](https://material.angular.io/)

<br />
### Instalación

Para hacer uso del design system primero hay que instalar la libreria npm [sinco angular](https://www.npmjs.com/package/@sinco/angular) e importar la ruta de la libreria `node_modules/@sinco/angular/src/lib/styles/theme.scss` en angular.json
