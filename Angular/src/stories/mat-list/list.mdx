# List

Las listas de Angular Material son un conjunto de componentes predefinidos que permiten la creación de listas ordenadas o desordenadas en una aplicación web Angular. Estos componentes incluyen opciones de diseño, funcionalidad de selección, paginación y filtrado, entre otras características útiles. Las listas de Angular Material están diseñadas para ser personalizables y flexibles, lo que las hace una excelente opción para la creación de interfaces de usuario sofisticadas y atractivas.
