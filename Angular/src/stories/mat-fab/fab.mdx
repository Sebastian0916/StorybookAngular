# FAB BUTTON

Los botones FAB (Floating Action Button) de Angular Material son botones circulares flotantes que se usan para acciones frecuentes en una aplicación web de Angular, como agregar elementos o iniciar acciones importantes. Se pueden personalizar con diferentes opciones de tamaño, color y estilo.
