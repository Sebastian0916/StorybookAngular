# PROGRESS SPIINER

Los progress spinner de Angular Material son un tipo de componente que muestra un indicador de progreso circular en la interfaz de usuario de una aplicación Angular. Estos componentes son útiles para indicar al usuario que una tarea está en curso y que se está esperando una respuesta o resultado. Los progress spinner de Angular Material ofrecen diferentes opciones de configuración, como el tamaño, el color y la animación, lo que los hace muy flexibles y adaptables a diferentes diseños y necesidades. Además, estos componentes son fáciles de integrar en una aplicación Angular utilizando la biblioteca de componentes Angular Material, lo que facilita su implementación y personalización.
