# SLIDE TOGGLE

El slide toggle de Angular Material es un componente interactivo que
permite al usuario cambiar entre dos estados mediante un interruptor deslizante.
Es fácil de usar y personalizar, y se puede utilizar para activar/desactivar funcionalidades
o cambiar la visibilidad de contenido en una aplicación Angular.
