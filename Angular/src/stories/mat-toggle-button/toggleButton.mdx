# BUTTON TOGGLE

Los botones de alternancia (toggle) de Angular Material son un tipo de componente de botón que permite al usuario alternar entre dos estados: activado o desactivado. Los botones de alternancia tienen una apariencia diferente dependiendo del estado actual, lo que ayuda a los usuarios a comprender fácilmente el estado actual del botón. Estos botones son fáciles de implementar y personalizar en una aplicación Angular utilizando la biblioteca de componentes Angular Material.
