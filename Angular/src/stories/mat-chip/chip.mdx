# CHIP

Los chips de Angular Material son etiquetas pequeñas que se utilizan para mostrar información estructurada, como nombres de usuario, categorías o selecciones de opciones en una lista compacta y visible. Se pueden personalizar con diferentes opciones de color, forma y estilo para adaptarse a la apariencia y la funcionalidad de la aplicación web de Angular.
