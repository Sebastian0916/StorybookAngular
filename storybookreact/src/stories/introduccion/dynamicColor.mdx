# Dynamic Color

##

"Hook personalizado que recibe como argumento la ruta absoluta de una imagen (en formato SVG) en la raíz
de la aplicación, actualizando las paletas `Primary`, `Secondary` y `Background` en los componentes de la
aplicación, basándose en la imagen proporcionada."

### Dependencias

- npm i @material/material-color-utilities

### Implementación

```tsx
import { useDynamicColor } from "@sinco/react";

const App = () => {
  const { loading } = useDynamicColor(" **/imagen.svg** ");

  return <>{loading && <> **Codigo de su aplicación** </>}</>;
};
```
