# Tipogrfia

<div class="mt-3"> 
Se añadieron 10 nuevas variantes de tipográfia global para estilizar etiquetas de tipo `<Typography>` y una clase de capitalización de texto en elementos: 
 > -  ```h3``` 
 > -  ```h4``` 
 > -  ```h5```      
 > -  ```h6```
 > -  ```body1```
 > -  ```body2```
 > -  ```body3```
 > -  ```subtitle1```
 > -  ```subtitle2```
 > -  ```caption```
 > -  ```overline```

**Font family:** Roboto y Nunito

**Weights:** 200, 300, 400, 500, 600, 700, 800, 900, 1000

<br/>
Cada módulo tiene su propia configuración de unidades de medida y clases de estilo para ajustar el color del texto, las cuales utilizan variables propias. Todo esto se explica en la sección de `clases de texto`.
</div>
